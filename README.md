# React chatbot using React and Gemini API

A modern AI chat application inspired by Google's Gemini, built with **React** and powered by the **Google Gemini API**. The project provides a clean, responsive, and interactive interface that allows users to communicate with an AI assistant in real time.

## Features

*  Real-time AI conversations using the Gemini API
*  Fast and responsive React-based user interface
*  Clean and intuitive chat experience
*  Fully responsive design for desktop and mobile devices
*  Modern UI with smooth interactions
*  Loading states and error handling for a better user experience
*  Component-based architecture for easy maintenance and scalability

## Technologies Used

* React
* JavaScript (ES6+)
* Google Gemini API
* CSS / Tailwind CSS 
* Vite
